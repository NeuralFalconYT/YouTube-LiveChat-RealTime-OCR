# Assets Folder

This folder contains optional example assets used by the demo script (`echo_bot.py`):

### ✔ Sample Background Images
These images are included only as **demonstration backgrounds** for rendering the VTuber-style chat display.  
Creators can replace them with their own artwork.

### ✔ Font File
The included font file is used for text rendering inside the example VTuber frame.

## ⚠ Important Notes
- These assets are **not required** for the OCR system itself.
- They are included only to help users test the example TTS bot quickly.
