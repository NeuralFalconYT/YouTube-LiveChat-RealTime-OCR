# RapidOCR Model Source (Apache License 2.0)

The OCR model files in this folder are downloaded from the official RapidOCR distribution sources.

## 📥 Download Sources

### ModelScope (Primary Source)
These models were downloaded from the RapidAI ModelScope repository:

https://www.modelscope.cn/models/RapidAI/RapidOCR/tree/master/onnx

### GitHub (Alternate Source)
RapidOCR models are also available on GitHub:

https://github.com/RapidAI/RapidOCR

Both sources distribute the same Apache-2.0 licensed PP-OCRv5 ONNX models.

---

## 📦 Models Included in This Repository

- `ch_PP-OCRv5_mobile_det.onnx`  
  Text detection model (PP-OCRv5)

- `en_PP-OCRv5_rec_mobile_infer.onnx`  
  English text recognition model (PP-OCRv5)

No modifications have been made to these files.
